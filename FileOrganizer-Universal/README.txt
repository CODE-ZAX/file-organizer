# File Organizer - Universal Package

This package contains File Organizer for all supported platforms.

## Installation

### Windows
1. Navigate to the `Windows` folder
2. Run `FileOrganizer.exe`

### macOS
1. Navigate to the `macOS` folder
2. Open `FileOrganizer.app`

### Linux
1. Navigate to the `Linux` folder
2. Run `./file-organizer`

## Requirements

- Windows 10 or later
- macOS 10.14 or later
- Linux with GTK+ (most modern distributions)
